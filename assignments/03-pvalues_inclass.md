# P-values | Week 03 In-class assignment

This assignment is designed to help you build a hands-on, ground-up understanding
of p-values by implementing a permutation test from scratch — computing a p-value
without invoking any formula or distributional assumption.

The notebook walks through a step-by-step permutation test on a simulated
clinical trial (placebo vs. drug), and ends by comparing the result to what a
standard t-test gives.

Here's what to do:

1. Download the [R notebook](./03-pvalues_inclass.R) or
   [Python notebook](./03-pvalues_inclass.py)
2. Work with a partner to go through the code step by step — for each block,
   discuss what it's doing before moving on
3. Add detailed comments at every step explaining both *what* the code does and
   *what it means statistically* — annotate as if writing for someone who has
   never heard of a permutation test
4. Run the code block by block and talk through the output at each stage
5. Answer the two questions at the end of the notebook
