# Statistical power | Week 04 Homework assignment

This assignment extends the in-class power analysis to multiple sample sizes,
so you can see how power depends on both effect size and the number of
observations you collect.

Write your answers directly in the script as comments, export as a PDF, and
submit using [this form](https://forms.gle/977e4vrVshwTyTcK7).

1. Download the [R script](./04-power_homework.R) or [Python script](./04-power_homework.py)
2. Add detailed comments at every step explaining both *what* the code does and *what it means statistically*
3. Run the code and answer all questions in the spaces provided

**Questions:**

3. What are your observations on how the null distribution changes with sample size?

4. What are your interpretations of the multiple power curves? Write your thoughts in terms of the dependence of power on both effect size and sample size.

5. A national sports organisation wants to design an experiment to detect biased coins used in pre-game tosses. How would you use the power curves above to help them design the experiment? What questions would you need answered before making a specific sample-size recommendation?

6. If you make a specific sample-size recommendation, write down what you will convey to the organisation in terms of error rates (false-positive rate and false-negative rate).

7. Which parts of your reasoning and recommendations change if the organisation says it cannot tolerate more than 1 biased coin for every 100 coins it uses? (Hint: this corresponds to changing alpha from 0.05 to 0.01.)
